# IGdown
Skrip Downloader IG Simple Untuk Pemula 
## Install
```
$ pkg instal python
$ pkg instal nano
$ pkg instal git
$ pip install beautifulsoup4
$ pip2 install requests
$ pip2 install mechanize
$ git clone https://github.com/Aldi098/Free-Proxy
$ cd Proksi Gratis
$ python free-proxy.py
```
